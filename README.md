# Concert Alerts 🎶

התראות בטלגרם כשאמנים שאתה אוהב פותחים הופעות חדשות. רץ חינם על GitHub Actions.

## איך זה עובד
- כל שעה (חוץ משבת, שעון ישראל) GitHub Actions סורק את אתרי ההופעות.
- מזהה הופעות חדשות בהשוואה לסריקה הקודמת (`data/shows.json`).
- מוסיף אמנים חדשים לקטלוג (`data/artists.json`), בלי כפילויות.
- שולח התראה בטלגרם — רק על אמנים שסימנת שאתה עוקב אחריהם.

## הקמה חד-פעמית (~15 דקות)

### 1. צור בוט טלגרם
1. פתח צ'אט עם **@BotFather** בטלגרם, שלח `/newbot`, ובחר שם.
2. הוא יחזיר לך **token** (משהו כמו `123456:ABC...`). שמור אותו.

### 2. העלה את הפרויקט ל-GitHub
צור ריפו **פרטי** חדש ודחוף אליו את התיקייה הזו (או העלה דרך הדפדפן).

### 3. הוסף את הטוקן כסוד
ב-GitHub: **Settings → Secrets and variables → Actions → New repository secret**
- Name: `TELEGRAM_BOT_TOKEN`
- Secret: הטוקן מ-BotFather

### 4. הפעל Actions
לשונית **Actions** → אשר הרצת workflows. (אם הריפו פרטי, ההרצות בחינם עד 2000 דקות בחודש — יותר ממספיק.)

### 5. הרשם בבוט
שלח `/start` לבוט שלך בטלגרם. בריצה הבאה הוא יזהה אותך.

זהו. מכאן זה אוטומטי.

## שימוש יומיומי (בטלגרם)
- `/artists` — רשימת כל האמנים שהתגלו, עם סימון מי שאתה עוקב אחריו.
- `/follow <מספר או שם>` — התחל לעקוב.
- `/unfollow <מספר או שם>` — הפסק לעקוב.
- `/following` — מי שאתה עוקב אחריו כרגע.

> בחירות מעובדות בריצה הקרובה (עד שעה). לעדכון מיידי: לשונית **Actions → concert-scan → Run workflow**.

## הוספת אתר חדש
1. צור `scrapers/<site>.py` עם מחלקה שיורשת מ-`Scraper`, מגדירה `name`, ומחזירה רשימת `Show`.
2. הוסף `from . import <site>` ל-`scrapers/__init__.py`.
3. כשהאתרים האמיתיים מוכנים, כבה את הדמו: הוסף repository variable `EXAMPLE_SCRAPER=off`.

## בדיקה מקומית
```bash
pip install -r requirements.txt
SKIP_SATURDAY=false python scan.py    # רץ בלי טוקן, מדפיס במקום לשלוח
```
